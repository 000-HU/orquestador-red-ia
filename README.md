## 🚀 Arquitectura del Sistema
Este proyecto es un **Orquestador de Infraestructura Híbrida** diseñado para el monitoreo de redes de baja latencia con un consumo de recursos ultra-bajo (<35MB RAM).

- **Músculo (Data Plane):** Algoritmos de Teoría de Grafos y Recursión implementados en **Java**.
- **Cerebro (Control Plane):** Motor de orquestación, telemetría y visualización en **Node.js**.
- **Visualización:** Monitor de latencia con gráficas **ASCII 8-bit** en tiempo real.

## 📊 Visualización de Telemetría (CLI)
El sistema genera reportes visuales directamente en el buffer de la terminal:
